SA ID Validator

This project implements a validator for South African ID numbers following the Test-Driven Development (TDD) approach. The validation checks ensure that ID numbers conform to the official format: YYMMDDSSSSCAZ, where: YYMMDD: Date of birth SSSS: Gender identifier (0000-4999 for females, 5000-9999 for males) C: Citizenship status (0 for South African citizens, 1 for permanent residents) Z: Checksum digit (calculated using the Luhn algorithm)
